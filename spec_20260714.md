## 2. 신규 기능 영역

### 2.1 권한 / 접근 제어 (신규 도입)
- **마스터 / 부마스터** 체계: 마스터는 전체 메뉴 접근, 부마스터는 지정 메뉴에만 마스터 수준 권한.
- **부서별 메뉴 접근 권한**: 메뉴별 접근 가능한 부서 설정 (`MENU_PERMS` 정의).
- 접근 시 권한 체크 후 미허용 시 토스트로 차단 안내.
- 관련 함수: `canAccess`, `deptCanAccess`, `renderPermSettings`, `renderSidebarPerms`, `savePermSettings`, `toggleMaster*`, `toggleSubMaster`, `toggleMasterQuick`, `migrateMemberDepts`, `normalizeDept`.
- 저장소: `uni_menu_perms`.

### 2.2 환경질현황 / 환경 성적서 (`env`, `env-form`)

- 출장기록 폼, 계약 탭·수금 탭 연동, 금액 자동 계산.

## 5. 영향 / 리스크

- **단일 파일 SPA**: 모든 기능이 `index.html` 하나에 포함됨. 서버/빌드 단계 없음.
- **데이터 이전 없음**: 기존 사용자의 localStorage 에는 v0.9 도메인 데이터가 없으므로 초기 진입 시 샘플/빈 데이터로 시작. `uni_members` 는 그대로 유지되나 부서 정규화(`migrateMemberDepts`, `normalizeDept`)가 최초 로드시 동작.
- **권한 기본값 주의**: `uni_menu_perms` 미설정 상태에서의 기본 접근 정책을 배포 전 확인 필요 (마스터 계정 잠김 방지).
- **줄바꿈**: 저장소 정책상 LF→CRLF 변환 경고가 발생할 수 있으나 기능 영향 없음.
