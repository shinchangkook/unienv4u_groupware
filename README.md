## 🚀 바로 실행

### 방법 1) 데모 모드 (서버 없이)
```
index.html 파일을 브라우저에서 열면 바로 실행됩니다.
데이터는 브라우저 localStorage 에 저장됩니다(체험용).
```

### 방법 2) 서버 연동 모드 (실제 인증 + DB) — Phase 0~1
```bash
cd server
npm install
cp .env.example .env      # 최초 1회 (JWT_SECRET 무작위 값으로 변경 권장)
npm start                 # http://localhost:4000 접속
```
Node + Express + SQLite 백엔드로 **서버 인증(JWT + 비밀번호 해시)** 과 회원가입·승인·
로그인을 처리합니다. 자세한 내용은 [`server/README.md`](server/README.md) 와
구축 가이드 [`uni_build_20260714.md`](uni_build_20260714.md) 를 참고하세요.

## 📋 주요 기능

### 그룹웨어
- 📢 공지사항 / 자유게시판
- ✅ 전자결재 (기안 → 팀장 → 대표이사)
- 📅 일정 / 캘린더
- 📓 업무일지
- 🏢 조직도 / 직원 디렉토리

### 인사·근태관리
- 👤 사원등록 (영문이름, 주민번호, 개인이메일, 퇴사일, 인사발령)
- 📅 연차생성 (근로기준법 자동 계산)
- 🏖️ 휴가관리 (연차/월차/반차/병가 승인·반려)

### 사업관리
- 📄 계약·미수금관리 (진행률, 경과일 표시)
- 🌿 환경질현황표 (수질·대기·토양·소음)

### 수금관리
- 🏪 거래처등록 (파일 업로드 → 자동 반영)
- 🏷️ 계정과목 관리
- 📒 계정별원장 (D3 연도 자동추출, 연속 표시)
- 📊 거래처원장 — 잔액 / 내용 탭

### 장비관리
- 🔧 장비관리대장 (규격·기기번호·제작회사·주요용도)
- 🛠️ 고장·수리내역 (담당자·기술책임자 확인 서명)
- 🎓 검·교정 현황 (유효기간 자동 판정)

### 회원 시스템
- 🔐 로그인 / 회원가입 (승인 구조)
- 👑 마스터 관리자 — 가입 승인·거절, 마스터 권한 설정
- 📱 반응형 지원 (PC / 태블릿 / 모바일)
